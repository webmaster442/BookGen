Compresses/Minifies a HTML document
-------------------------------------------------------------------------------

    BookGen Md2HTML -i [input.html] -o [output.html]
    BookGen Md2HTML --input [input.html] --output [output.html]

    Arguments:
    -i, --input: 
        Input markdown file path

    -o, --output: 
        Output html file path. If file name is "con", outputs to console.
